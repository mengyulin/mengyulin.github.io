1. 教學目標

本講義說明如何使用 Fenton 所提出的直接迭代法，計算梯形渠道的正規水深 （normal depth）。此方法是將 Manning 公式重新整理為固定點迭代形式：

\[ y_{i+1}=F(y_i). \]

如此一來，只要給定初始值 \(y_0\)，便可重複代入同一公式，逐步求得正規水深。

本講義使用下列符號：

  • \(y\)：水深。
  • \(y_n\)：正規水深。
  • \(b\)：梯形渠道底寬。
  • \(B\)：水面寬。
  • \(m\)：單側邊坡，水平比垂直。
  • \(A\)：通水面積。
  • \(P\)：潤周。
  • \(R\)：水力半徑。
  • \(Q\)：流量。
  • \(n\)：Manning 粗糙係數。
  • \(S_0\)：底床坡度。

2. 正規水深問題

對一均勻流而言，水深稱為正規水深 \(y_n\)。

若已知：

\[ Q,\quad n,\quad S_0, \]

以及渠道斷面幾何關係 \(A(y)\)、\(P(y)\)，則 Manning 公式可寫成

\[ Q= \frac{1}{n}AR^{2/3}S_0^{1/2}. \]

其中

\[ R=\frac{A}{P}. \]

因此

\[ Q= \frac{1}{n} A \left(\frac{A}{P}\right)^{2/3} S_0^{1/2}. \]

整理得

\[ Q= \frac{1}{n} \frac{A^{5/3}}{P^{2/3}} S_0^{1/2}. \]

由於 \(A\) 與 \(P\) 都是水深 \(y\) 的函數，因此這是一個關於 \(y\) 的 非線性方程。正規水深 \(y_n\) 就是使此方程成立的水深。

3. Fenton 直接迭代法的想法

Fenton 方法的關鍵是將 Manning 公式改寫成容易迭代的形式。

先將通水面積 \(A\) 寫成

\[ A = y\left(\frac{A}{y}\right). \]

因此

\[ A^{5/3} = y^{5/3} \left(\frac{A}{y}\right)^{5/3}. \]

代入 Manning 公式：

\[ Q= \frac{1}{n} \frac{ y^{5/3} \left(\dfrac{A}{y}\right)^{5/3} } {P^{2/3}} S_0^{1/2}. \]

將 \(y^{5/3}\) 移到左邊，可得

\[ \frac{Q}{y^{5/3}} = \frac{1}{n} S_0^{1/2} \frac{ \left(\dfrac{A}{y}\right)^{5/3} } {P^{2/3}}. \]

接著將 \(y\) 解出，得到

\[ y = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{P(y)^{2/5}}{A(y)/y}. \]

因此可定義迭代公式：

\[ \boxed{ y_{i+1} = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{P(y_i)^{2/5}}{A(y_i)/y_i}. } \]

其中 \(y_{i}\) 是第 \(i\) 次的迭代值，這就是 Fenton 直接迭代法。

4. 為什麼這個方法通常會收斂很快？

在許多渠道中，尤其是寬度相對水深較大的渠道，\(A/y\) 可以視為一種 代表性寬度，而 \(P\) 也不會隨水深劇烈變化。

因此，在

\[ y_{i+1} = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{P(y_i)^{2/5}}{A(y_i)/y_i} \]

中，右側除了常數項之外，隨 \(y_i\) 變化的部分通常變化較緩慢。 這使得直接迭代法對一般梯形渠道常能快速收斂。

5. 初始值的選取

Fenton 建議先取一個代表性寬度，並用寬渠道近似估計初始水深。

為避免與實際水面寬 \(B\) 混淆，本講義將初始估計用的代表性寬度記為

\[ B_{\mathrm{est}}. \]

在寬渠道近似下：

\[ \frac{A}{y}\approx B_{\mathrm{est}}, \]

\[ P\approx B_{\mathrm{est}}. \]

將此近似代入 Fenton 迭代式，可得初始水深估計式：

\[ \boxed{ y_0 = \left( \frac{nQ} {B_{\mathrm{est}}\sqrt{S_0}} \right)^{3/5}. } \]

若沒有更好的估計，可以先取

\[ B_{\mathrm{est}}=b. \]

也就是用底寬作為代表性寬度。

6. 梯形渠道的幾何關係

對梯形渠道，若底寬為 \(b\)，單側邊坡為 \(m\)，水深為 \(y\)，則水面寬為

\[ B=b+2my. \]

通水面積為

\[ A=\frac{(b+B)y}{2}. \]

由於

\[ B=b+2my, \]

所以

\[ A=by+my^2. \]

也可寫成

\[ A=y(b+my). \]

因此

\[ \frac{A}{y}=b+my. \]

潤周為

\[ P=b+2y\sqrt{1+m^2}. \]

7. 梯形渠道的 Fenton 迭代式

將梯形渠道的幾何關係代入 Fenton 公式：

\[ y_{i+1} = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{P(y_i)^{2/5}}{A(y_i)/y_i}. \]

由於

\[ P(y_i)=b+2y_i\sqrt{1+m^2}, \]

且

\[ \frac{A(y_i)}{y_i}=b+my_i, \]

所以梯形渠道的迭代式為

\[ \boxed{ y_{i+1} = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{ \left[b+2y_i\sqrt{1+m^2}\right]^{2/5} } {b+my_i}. } \]

為簡化計算，可先定義常數

\[ C= \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5}. \]

則

\[ y_{i+1} = C \frac{ \left[b+2y_i\sqrt{1+m^2}\right]^{2/5} } {b+my_i}. \]

8. 計算流程整理

使用一般計算機時，可按照下列步驟：

  1. 寫出梯形渠道幾何式：

    \[ A=y(b+my), \]

    \[ P=b+2y\sqrt{1+m^2}. \]

  2. 計算常數：

    \[ C= \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5}. \]

  3. 寫出迭代式：

    \[ y_{i+1} = C \frac{ \left[b+2y_i\sqrt{1+m^2}\right]^{2/5} } {b+my_i}. \]

  4. 選擇代表性寬度 \(B_{\mathrm{est}}\)，並計算初始值：

    \[ y_0 = \left( \frac{nQ} {B_{\mathrm{est}}\sqrt{S_0}} \right)^{3/5}. \]

  5. 若無其他資料，可先取

    \[ B_{\mathrm{est}}=b. \]

  6. 將 \(y_0\) 代入迭代式，求得 \(y_1\)。
  7. 再將 \(y_1\) 代入迭代式，求得 \(y_2\)。

  8. 重複計算：

    \[ y_0 \rightarrow y_1 \rightarrow y_2 \rightarrow y_3 \rightarrow \cdots \]

  9. 當相鄰兩次結果差異很小，例如

    \[ \vert y_{i+1}-y_i \vert < 0.001~\mathrm{m}, \]

    即可停止，並取該值為正規水深 \(y_n\)。

9. 例題

一梯形渠道之底寬為

\[ b=3.0~\mathrm{m}, \]

邊坡為

\[ m=1.5, \]

也就是水平：垂直為 \(1.5:1\)。渠道底床坡度為

\[ S_0=0.0016, \]

Manning 粗糙係數為

\[ n=0.015, \]

流量為

\[ Q=15.0~\mathrm{m^3/s}. \]

試使用 Fenton 直接迭代法求正規水深 \(y_n\)。

10. 例題解答

10.1. Step 1：寫出梯形渠道幾何式

本題中

\[ b=3.0, \qquad m=1.5. \]

水面寬為

\[ B=3.0+2(1.5)y, \]

即

\[ B=3.0+3.0y. \]

通水面積為

\[ A=y(b+my), \]

因此

\[ A=y(3.0+1.5y). \]

所以

\[ \frac{A}{y}=3.0+1.5y. \]

潤周為

\[ P=b+2y\sqrt{1+m^2}. \]

代入 \(b=3.0\)、\(m=1.5\)：

\[ P=3.0+2y\sqrt{1+1.5^2}. \]

因為

\[ \sqrt{1+1.5^2} = \sqrt{3.25} \approx 1.8028, \]

所以

\[ P=3.0+3.6056y. \]

10.2. Step 2：計算常數 \(C\)

\[ C= \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5}. \]

代入

\[ n=0.015, \qquad Q=15.0, \qquad S_0=0.0016. \]

因為

\[ \sqrt{S_0}=\sqrt{0.0016}=0.04, \]

所以

\[ C= \left( \frac{0.015(15.0)}{0.04} \right)^{3/5}. \]

\[ C=(5.625)^{3/5}. \]

計算得

\[ C\approx 2.8189. \]

10.3. Step 3：寫出本題的迭代公式

\[ y_{i+1} = C \frac{ \left[b+2y_i\sqrt{1+m^2}\right]^{2/5} } {b+my_i}. \]

代入本題數值：

\[ y_{i+1} = 2.8189 \frac{ (3.0+3.6056y_i)^{2/5} } {3.0+1.5y_i}. \]

10.4. Step 4：選取初始值

取代表性寬度為底寬：

\[ B_{\mathrm{est}}=b=3.0~\mathrm{m}. \]

因此

\[ y_0 = \left( \frac{nQ} {B_{\mathrm{est}}\sqrt{S_0}} \right)^{3/5}. \]

代入：

\[ y_0 = \left( \frac{0.015(15.0)} {3.0(0.04)} \right)^{3/5}. \]

\[ y_0 = (1.875)^{3/5}. \]

計算得

\[ y_0\approx 1.4581~\mathrm{m}. \]

10.5. Step 5：逐步迭代

使用

\[ y_{i+1} = 2.8189 \frac{ (3.0+3.6056y_i)^{2/5} } {3.0+1.5y_i}. \]

由

\[ y_0=1.4581~\mathrm{m} \]

開始計算。

11. 與 try and error 方法的比較

前一份講義中的 try and error 方法是：

\[ \text{猜 } y \rightarrow \text{算 } A,P,R \rightarrow \text{算 } Q_M \rightarrow \text{比較 } Q_M \text{ 與 } Q. \]

若

\[ Q_M

則表示水深太小；若

\[ Q_M>Q, \]

則表示水深太大。接著需要靠人工判斷逐步縮小水深範圍。

Fenton 方法則是：

\[ \text{給定 } y_i \rightarrow \text{代入迭代式} \rightarrow \text{得到 } y_{i+1}. \]

因此它不需要每一步判斷水深太大或太小，而是直接重複使用同一個公式， 直到相鄰兩次的結果幾乎相同。

12. 學習提醒

  1. 初始估計式中的 \(B_{\mathrm{est}}\) 是代表性寬度，不一定等於實際水面寬。
  2. Fenton 方法是固定點迭代法。對一般梯形渠道通常收斂很快，但仍應檢查 \[ \vert y_{i+1}-y_i \vert \] 是否逐漸變小。
  3. 若迭代結果振盪或發散，應改用 try and error、二分法或 Newton 法等其他數值方法。

13. 結論

Fenton 直接迭代法將 Manning 正規水深問題整理成

\[ y_{i+1}=F(y_i), \]

其中對梯形渠道：

\[ y_{i+1} = \left( \frac{nQ}{\sqrt{S_0}} \right)^{3/5} \frac{ \left[b+2y_i\sqrt{1+m^2}\right]^{2/5} } {b+my_i}. \]

此方法的優點是計算流程固定，適合使用一般計算機反覆代入。
