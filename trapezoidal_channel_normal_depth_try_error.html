1. 教學目標

本講義說明如何使用一般計算機，以試誤法（try and error）或二分法 （bisection method），由 Manning 公式求梯形渠道的正規水深 （normal depth）。

在渠道水力學中，若已知流量 \(Q\)、渠道斷面形狀、Manning 粗糙係數 \(n\) 與底床坡度 \(S_0\)，要求正規水深 \(y_n\)，通常必須解一個 非線性方程。若沒有電腦或可程式計算機，也可以利用一般計算機逐步試算。

本講義使用下列符號：

  • \(b\)：梯形渠道底寬。
  • \(B\)：水面寬。
  • \(m\)：單側邊坡，水平比垂直。
  • \(A\)：通水面積。
  • \(P\)：潤周。
  • \(R\)：水力半徑。
  • \(Q\)：已知流量。
  • \(Q_M(y)\)：由 Manning 公式對某一試算水深 \(y\) 所算出的流量。
  • \(S_0\)：底床坡度。
  • \(n\)：Manning 粗糙係數。
  • \(y_n\)：正規水深。

2. 題目

一梯形渠道之底寬為

\[ b=3.0~\mathrm{m}, \]

邊坡為

\[ m=1.5, \]

也就是水平：垂直為 \(1.5:1\)。渠道底床坡度為

\[ S_0=0.0016, \]

Manning 粗糙係數為

\[ n=0.015, \]

流量為

\[ Q=15.0~\mathrm{m^3/s}. \]

試使用一般計算機，以試誤法求正規水深 \(y_n\)。

3. 梯形渠道的幾何關係

若水深為 \(y\)，則水面寬為

\[ B=b+2my. \]

其中：

  • \(b\) 為底寬。
  • \(B\) 為水面寬。
  • \(m\) 為單側邊坡，表示水平距離與垂直距離之比。

本題中

\[ b=3.0, \qquad m=1.5, \]

所以

\[ B=3.0+2(1.5)y, \]

也就是

\[ B=3.0+3.0y. \]

通水面積為梯形面積：

\[ A=\frac{(b+B)y}{2}. \]

由於

\[ B=b+2my, \]

所以通水面積也可寫成

\[ A=by+my^2. \]

本題中

\[ A=3.0y+1.5y^2. \]

潤周為

\[ P=b+2y\sqrt{1+m^2}. \]

本題中

\[ P=3.0+2y\sqrt{1+1.5^2}. \]

水力半徑為

\[ R=\frac{A}{P}. \]

4. 公制 Manning 公式

公制 Manning 公式為

\[ Q= \frac{1}{n}AR^{2/3}S_0^{1/2}. \]

正規水深 \(y_n\) 是使 Manning 公式算出的流量等於已知流量的水深。

因此，對某一個試算水深 \(y\)，定義

\[ Q_M(y)= \frac{1}{n}A(y)R(y)^{2/3}S_0^{1/2}. \]

我們要找的是

\[ Q_M(y)=Q. \]

也可寫成求根形式：

\[ f(y)=Q_M(y)-Q. \]

正規水深 \(y_n\) 即為使

\[ f(y)=0 \]

的水深。

本題中，因為

\[ Q=15.0~\mathrm{m^3/s}, \]

所以

\[ f(y)=Q_M(y)-15.0. \]

5. 試誤法的基本判斷

在一般渠道中，當水深 \(y\) 增加時，通水面積 \(A\) 會增加， 水力半徑 \(R\) 通常也會增加，因此由 Manning 公式算出的流量 \(Q_M(y)\) 也會增加。

因此可用下列方式判斷：

  • 若 \(Q_M(y) < Q\)，表示試算水深太小。
  • 若 \(Q_M(y) > Q\)，表示試算水深太大。
  • 若 \(Q_M(y) \approx Q\)，表示該水深接近正規水深。

也就是說：

\[ f(y) < 0 \quad\Rightarrow\quad \text{水深太小}, \]

\[ f(y) > 0 \quad\Rightarrow\quad \text{水深太大}. \]

6. 第一次粗略試算

先試幾個水深，計算 \(B\)、\(A\)、\(P\)、\(R\)、\(Q_M\) 與 \(f(y)\)。

Table 1: 第一次粗略試算
\(y\) (m) \(B\) (m) \(A\) (\(\mathrm{m^2}\)) \(R\) (m) \(Q_M\) (\(\mathrm{m^3/s}\)) \(f(y)=Q_M-Q\)
1.20 6.60 5.760 0.786 13.084 -1.916
1.30 6.90 6.435 0.837 15.242 +0.242
1.40 7.20 7.140 0.887 17.580 +2.580

由表中可知

\[ f(1.20) < 0, \]

而

\[ f(1.30) > 0. \]

所以正規水深位於

\[ 1.20 < y_n < 1.30. \]

也就是說，根被夾在 1.20 m 與 1.30 m 之間。

7. 縮小範圍

接著試中間值

\[ y=1.25~\mathrm{m}. \]

Table 2: 縮小根所在區間
\(y\) (m) \(Q_M\) (\(\mathrm{m^3/s}\)) \(f(y)=Q_M-15.0\)
1.20 13.084 -1.916
1.25 14.141 -0.859
1.30 15.242 +0.242

因為

\[ f(1.25) < 0, \]

而

\[ f(1.30) > 0, \]

所以根位於

\[ 1.25 < y_n < 1.30. \]

8. 繼續用二分法逼近

繼續在目前的區間內試算。每次選擇區間中間附近的水深，重新計算 \(B\)、\(A\)、\(P\)、\(R\)、\(Q_M\) 與 \(f(y)\)，並根據 \(f(y)\) 的正負號縮小根所在範圍。

Table 3: 進一步逼近正規水深
\(y\) (m) \(Q_M\) (\(\mathrm{m^3/s}\)) \(f(y)=Q_M-15.0\) 判斷
1.2750 14.686 -0.314 水深太小
1.2875 14.962 -0.038 水深太小
1.2938 15.102 +0.102 水深太大
1.2900 15.018 +0.018 水深稍大
1.2888 14.990 -0.010 水深稍小
1.2894 15.004 +0.004 已接近
1.2891 14.997 -0.003 已接近
1.2892 15.000 約 0 可接受

因此可取

\[ \boxed{ y_n \approx 1.289~\mathrm{m} } \]

作為本題的正規水深。

9. 給學生的計算流程

使用一般計算機時，可按照以下步驟：

  1. 假設一個水深 \(y\)。

  2. 計算水面寬：

    \[ B=b+2my. \]

  3. 計算通水面積：

    \[ A=\frac{(b+B)y}{2}, \]

    或使用等價形式

    \[ A=by+my^2. \]

  4. 計算潤周：

    \[ P=b+2y\sqrt{1+m^2}. \]

  5. 計算水力半徑：

    \[ R=\frac{A}{P}. \]

  6. 用 Manning 公式計算流量：

    \[ Q_M= \frac{1}{n}AR^{2/3}S_0^{1/2}. \]

  7. 比較 \(Q_M\) 與已知流量 \(Q\)：

    \[ f(y)=Q_M-Q. \]

  8. 若 \(f(y) < 0\)，表示水深太小。
  9. 若 \(f(y) > 0\)，表示水深太大。
  10. 找到 \(f(y)\) 正負號改變的區間後，繼續取中間值逼近。
  11. 當 \(Q_M\) 已接近 \(Q\)，或 \(f(y)\) 已接近 0 時，該水深即為正規水深。

10. 與其他求根問題的比較

在渠道水力學中，常見需要用試誤法或數值方法求根的問題包括：

Table 4: 渠道水力學常見求根問題
問題 要求的未知量 求根方程式
Alternate depth \(y\) \(E(y)-E_1=0\)
Critical depth \(y_c\) \(\dfrac{Q^2B}{gA^3}-1=0\)
Normal depth \(y_n\) \(Q_M(y)-Q=0\)

其中正規水深問題的求解重點是：

\[ \boxed{ \text{猜水深 } y \rightarrow \text{算 } A,P,R \rightarrow \text{算 } Q_M \rightarrow \text{比較 } Q_M \text{ 與 } Q } \]

11. 教學提醒

  1. 公制 Manning 公式為

    \[ Q= \frac{1}{n}AR^{2/3}S_0^{1/2}. \]

    不需要乘上英制係數 1.49。

  2. 每次試算新的水深 \(y\) 時，都必須重新計算：

    \[ B,\quad A,\quad P,\quad R,\quad Q_M. \]

  3. 梯形渠道中，水面寬為

    \[ B=b+2my, \]

    不是常數。

  4. 潤周不是 \(b+2y\)，而是

    \[ P=b+2y\sqrt{1+m^2}. \]

    其中 \(\sqrt{1+m^2}\) 是邊坡斜邊長與垂直水深的比例。

  5. 若 \(Q_M < Q\)，表示水深太小；若 \(Q_M > Q\)，表示水深太大。

12. 常見錯誤

  1. 將梯形渠道的水面寬 \(B\) 誤認為底寬 \(b\)。
  2. 將潤周誤寫為 \(P=b+2y\)，這是矩形渠道的公式，不適用於梯形渠道。
  3. 忘記 Manning 公式中的水力半徑指數是 \(2/3\)。
  4. 在公制公式中誤乘英制係數 1.49。
  5. 每次改變水深時，沒有重新計算 \(A\)、\(P\) 與 \(R\)。
  6. 單位不一致。本題使用公制，因此 \[ Q: \mathrm{m^3/s},\quad b,B,y: \mathrm{m},\quad A: \mathrm{m^2},\quad S_0: \text{無因次}. \]

13. 課後練習

一梯形渠道底寬為

\[ b=4.0~\mathrm{m}, \]

邊坡為

\[ m=2.0, \]

底床坡度為

\[ S_0=0.0010, \]

Manning 粗糙係數為

\[ n=0.018, \]

流量為

\[ Q=20.0~\mathrm{m^3/s}. \]

請使用一般計算機，以試誤法求其正規水深 \(y_n\)。

提示：

  1. 先寫出

    \[ B=b+2my. \]

  2. 再寫出

    \[ A=by+my^2. \]

  3. 計算

    \[ P=b+2y\sqrt{1+m^2}. \]

  4. 計算

    \[ R=\frac{A}{P}. \]

  5. 建立

    \[ f(y)= \frac{1}{n}AR^{2/3}S_0^{1/2}-Q. \]

  6. 找出 \(f(y)\) 由負變正的區間，再逐步縮小。
